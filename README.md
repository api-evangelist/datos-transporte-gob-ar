# Datos Transporte Argentina (datos-transporte-gob-ar)

Datos Transporte Argentina is a government agency open-data portal for Argentina running **CKAN**, exposing a standardized catalog API over ~48 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/datos-transporte-gob-ar/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=datos-transporte-gob-ar-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.7.6

## API
- **Datos Transporte Argentina CKAN Action API** — base `https://datos.transporte.gob.ar/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://datos.transporte.gob.ar)
- Live endpoint verified: `https://datos.transporte.gob.ar/api/3/action/status_show` · ~48 datasets
